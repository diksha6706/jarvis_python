# 🤖 Jarvis AI Assistant

A voice-controlled virtual assistant built with Python that can listen, respond, search, and automate everyday tasks.

### Features
- 🎙️ Speech Recognition
- 🗣️ Text-to-Speech
- 🌐 Web Search
- 🎵 Music Playback
- ⚡ Task Automation

### Tech Stack
Python • SpeechRecognition • pyttsx3 • Automation

> Turning voice commands into actions.
